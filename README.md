# Описание
Python реализация бота для сайта https://lois.pro/ 
## Установка
Требуется python 3 версии
```bash
pip install -r requirements.txt
```
## Использование
```bash
1) cp .env.test .env
2) Заполнить поле PHONE в .env файле
3) Создать папку sessions ( mkdir sessions ) 
3) python main.py
```
